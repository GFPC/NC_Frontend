# Деплой на Vercel

## Инструкция по развертыванию

### 1. Подготовка

Убедитесь что бэкенд развернут на Render.com по адресу:
```
https://nc-backend-1hc2.onrender.com
```

### 2. Настройка переменных окружения в Vercel

В настройках проекта Vercel добавьте переменную окружения:

```
VITE_API_URL=https://nc-backend-1hc2.onrender.com/api
```

### 3. Деплой

Vercel автоматически определит настройки из `vercel.json`:
- Команда сборки: `npm run build:vercel`
- Выходная директория: `dist/public`

### 4. Проверка

После деплоя приложение должно:
- Загружать интерфейс кинозала
- Успешно подключаться к API бэкенда
- Позволять выбирать и бронировать места

## Структура проекта для Vercel

Vercel деплоит только фронтенд:
- ✅ `client/` - React приложение
- ✅ `dist/public/` - скомпилированные файлы
- ❌ `backend/` - игнорируется (деплоится на Render)
- ❌ `server/` - игнорируется (Express не нужен для Vercel)

## Отладка

Если после деплоя скачивается файл вместо отображения приложения:
1. Проверьте что `vercel.json` корректно настроен
2. Убедитесь что команда сборки `npm run build:vercel` выполняется успешно
3. Проверьте что переменная окружения `VITE_API_URL` установлена в Vercel
4. Откройте DevTools браузера и проверьте ошибки в консоли
